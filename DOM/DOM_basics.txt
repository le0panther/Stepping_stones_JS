Going Up ⬆️

.parentNode — parent node of any type
.parentElement — parent only if it's an element

Going Down ⬇️

.firstElementChild — first child element
.lastElementChild — last child element
.children — all direct child elements (HTMLCollection),indexing ke through isme hum elements access kar sakte hai 
.childNodes — all child nodes including text/comments (NodeList)

Going Sideways ↔️

.nextElementSibling — next sibling element
.previousElementSibling — previous sibling element
.nextSibling — next sibling node of any type
.previousSibling — previous sibling node of any type
